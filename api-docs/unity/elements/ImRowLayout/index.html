ceramic.Entity → ceramic.Visual → ceramic.Quad → ceramic.Layer → ceramic.View → ceramic.LinearLayout → ceramic.RowLayout → elements.ImRowLayout (Class)

A specialized row layout for the immediate mode UI system.

ImRowLayout extends the standard RowLayout with pre-configured settings optimized for immediate mode UI rendering. It provides consistent spacing between UI elements when they are arranged horizontally in a row.

This layout is automatically created and managed by the Im system when using row-based layouts through Im.beginRow() and Im.endRow().
