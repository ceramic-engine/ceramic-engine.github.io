ceramic.MimeType (Class)

MIME type utility class for determining content type characteristics.

This class provides helper methods to analyze MIME types and determine whether content should be treated as text or binary data. It's used internally by the HTTP plugin to decide how to process response content.

The classification is based on standard MIME type conventions and includes support for common text-based formats like HTML, CSS, JavaScript, JSON, XML, and various text subtypes.

Example usage:

var contentType = "application/json; charset=utf-8";

if (MimeType.isText(contentType)) {
    // Process as text content
    var textData = response.content;
} else {
    // Process as binary content
    var binaryData = response.binaryContent;
}

isBinary(type: String): Bool#

Determines if a MIME type represents binary content.

This method analyzes the given MIME type string and returns true if the content should be treated as binary data. The method handles MIME types with parameters (e.g., "text/html; charset=utf-8") by ignoring everything after the semicolon.

Text-based MIME types include:

  • All types starting with "text/" (e.g., text/plain, text/html)
  • JavaScript and JSON (application/javascript, application/json)
  • XML-based formats (application/xml, image/svg+xml, etc.)
  • Rich Text Format (application/rtf)
  • Perl scripts (application/x-perl)
  • Various specialized text formats

isText(type: String): Bool#

Determines if a MIME type represents text content.

This is a convenience method that returns the inverse of isBinary(). Use this when you want to check if content should be processed as text.
