# Ceramic API Documentation

Choose a target platform below to explore the available APIs.

<p class="extra-info">Most of the APIs are available on all targets. Choosing a target allows you to see what's available in addition to the cross-platform APIs.</p>

## Available Targets

| Target | Description |
|--------|-------------|
| [Clay (Native)](/api-docs/clay-native/) | Native desktop and mobile builds using the Clay backend |
| [Clay (Web)](/api-docs/clay-web/) | Web builds using the Clay backend with WebGL rendering |
| [Headless](/api-docs/headless/) | Server-side and headless builds without rendering |
| [Unity](/api-docs/unity/) | Unity integration for using Ceramic within Unity projects |
