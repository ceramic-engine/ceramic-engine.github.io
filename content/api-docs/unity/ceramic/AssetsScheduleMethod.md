# AssetsScheduleMethod

<div class="view-source"><a href="https://github.com/ceramic-engine/ceramic/blob/master/runtime/src/ceramic/AssetsScheduleMethod.hx">View source</a></div>

<div class="type-hierarchy"><strong>ceramic.AssetsScheduleMethod</strong> (Abstract)</div>

Defines how multiple assets are scheduled for loading.

This setting controls whether assets load simultaneously or sequentially,
affecting loading performance and resource usage.
