# Component

<div class="view-source"><a href="https://github.com/ceramic-engine/ceramic/blob/master/plugins/unity/runtime/src/unityengine/Component.hx">View source</a></div>

<div class="type-hierarchy"><a href="/api-docs/unity/unityengine/Object/">Object</a> → <strong>unityengine.Component</strong> (extern class) → <a href="/api-docs/unity/unityengine/Behaviour/">Behaviour</a></div>

Unity Component class extern binding for Ceramic.
Base class for all components that can be attached to GameObjects.

Components are the functional pieces of every GameObject.
This is a minimal binding that serves as the base class for
other Unity component types used by the Ceramic backend.
