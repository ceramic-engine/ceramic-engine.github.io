# MixBlend

<div class="view-source"><a href="https://github.com/jeremyfa/spine-hx/blob/master/spine/Animation.hx">View source</a></div>

<div class="type-hierarchy"><strong>spine.MixBlend</strong> (Abstract)</div>

Controls how timeline values are mixed with setup pose values or current pose values when a timeline is applied with
<code>alpha</code> < 1.

See Timeline {@link Timeline#apply(Skeleton, float, float, Array, float, MixBlend, MixDirection)}.
