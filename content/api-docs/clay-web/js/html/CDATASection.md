# CDATASection

<div class="type-hierarchy"><a href="/api-docs/clay-web/js/html/EventTarget/">EventTarget</a> → <a href="/api-docs/clay-web/js/html/Node/">Node</a> → <a href="/api-docs/clay-web/js/html/CharacterData/">CharacterData</a> → <a href="/api-docs/clay-web/js/html/Text/">Text</a> → <strong>js.html.CDATASection</strong> (extern class)</div>

The `CDATASection` interface represents a CDATA section that can be used within XML to include extended portions of unescaped text. The symbols `` and `&amp;` don’t need escaping as they normally do when inside a CDATA section.
