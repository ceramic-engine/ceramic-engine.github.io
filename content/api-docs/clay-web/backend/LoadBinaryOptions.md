# LoadBinaryOptions

<div class="view-source"><a href="https://github.com/ceramic-engine/ceramic/blob/master/plugins/clay/runtime/src/backend/LoadBinaryOptions.hx">View source</a></div>

<div class="type-hierarchy"><strong>backend.LoadBinaryOptions</strong> (Typedef)</div>

Configuration options for loading binary data in the Clay backend.

These options control how binary files are loaded, allowing
developers to choose between synchronous and asynchronous loading
based on their needs.
