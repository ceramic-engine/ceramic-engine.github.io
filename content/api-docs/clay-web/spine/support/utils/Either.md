# Either

<div class="view-source"><a href="https://github.com/jeremyfa/spine-hx/blob/master/spine/support/utils/Either.hx">View source</a></div>

<div class="type-hierarchy"><strong>spine.support.utils.Either</strong> (Abstract)</div>

Useful to limit a Dynamic function argument's type to the specified
type parameters. This does NOT make the use of Dynamic type-safe in
any way (the underlying type is still Dynamic and Std.isOfType() checks +
casts are necessary).
