# LoadTextureOptions

<div class="view-source"><a href="https://github.com/ceramic-engine/ceramic/blob/master/plugins/clay/runtime/src/backend/LoadTextureOptions.hx">View source</a></div>

<div class="type-hierarchy"><strong>backend.LoadTextureOptions</strong> (Typedef)</div>

Configuration options for loading texture/image resources in the Clay backend.

These options control how images are loaded and processed, including
alpha channel handling which is important for correct rendering.
