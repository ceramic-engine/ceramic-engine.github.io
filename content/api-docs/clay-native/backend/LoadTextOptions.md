# LoadTextOptions

<div class="view-source"><a href="https://github.com/ceramic-engine/ceramic/blob/master/plugins/clay/runtime/src/backend/LoadTextOptions.hx">View source</a></div>

<div class="type-hierarchy"><strong>backend.LoadTextOptions</strong> (Typedef)</div>

Configuration options for loading text files in the Clay backend.

These options control how text files are loaded, supporting both
synchronous and asynchronous loading patterns for different use cases.
