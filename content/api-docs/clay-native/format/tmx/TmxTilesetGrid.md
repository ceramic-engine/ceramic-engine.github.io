# TmxTilesetGrid

<div class="view-source"><a href="https://github.com/ceramic-engine/format-tiled/blob/master/format/tmx/Data.hx">View source</a></div>

<div class="type-hierarchy"><strong>format.tmx.TmxTilesetGrid</strong> (Class)</div>

This element is only used in case of isometric orientation, and determines how tile overlays for terrain and collision information are rendered.
