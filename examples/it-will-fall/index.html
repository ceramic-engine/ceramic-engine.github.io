It will Fall (LD49)

Full page — Source on GitHub — Ludum Dare

This jam game is a COMPO entry created by Jérémy Faivre, made for Ludum Dare 49 with the theme « Unstable ». It's a game made in a day that features 2D physics with nape. It also shows an example use case for ceramic.StateMachine.
